문자열 포맷팅의 새로운 방식 f-string
group = "샤이니"
name = "김종현" 
print(f"제가 좋아하는 가수 그룹은 {group]이고 제일 좋아하는 멤버는 {name}입니다.")


불 대수 : 일상적인 논리를 수학적으로 표현한 것(진리값)
           -> True, False
           -> AND, OR, NOT
  1)명제 : 참 또는 거짓이 명확한 문장
 * 두 개의 조건이 있을 경우
  2)AND : 둘 다 모두 참이거나 거짓이어야 한다.
  3)OR : 한쪽이 참이거나 거짓이거나 상관없다.
  4)NOT : 거짓의 값을 참으로, 참의 값을 거짓으로 변경해 준다.

불린형 : Python에서 참과 거짓을 표현하는 자료형
           -> 따옴표를 사용하지 않아야 한다.


type 함수 : 어떠한 자료형을 갖고 있는지 쉽게 알 수 있다.
