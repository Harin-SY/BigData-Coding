파이썬에서 #(코멘트)를 사용하는 이유는?
 -> "#"를 실행할 경우, 아무것도 출력되지 않는다
     1. 메모 능력 - 복잡한 코드 설명, 나중에 재확인하기 위한 목적, 다른 개발자와의 소통

 * 코멘트는 중요하다!

프로그래밍이란?
 -> 계산을 해야 되는 수식을 컴퓨터에게 알려주는 것
     Python을 사용하게 되면 "자료형(문자열)", "숫자(정수, 소수)", 불린(참,거짓) 이외에도 다양한 값을 사용할 수 있다.
 
추상화란?
-> 복잡한 내용은 숨기고 주요 기능만 집중하는 것이다.
     * 변수(Variable)
        -> 값을 저장해 준다.
            등호 : 지정 연산자
            price = 500
            500원이라는 값을 price라는 변수에 넣는 것이다.
            다른 사람이 봐도 알 수 있게 변수를 지정하는 것이 중요하다.
     * 함수(Function)
        -> 명령을 저장하는 것
             내장함수 : 자주 쓰이는 함수를 미리 구현해 둔 것
             def : 새로운 함수를 정의하는 것
             def love(): ->함수 정의(헤더)
                 print("Hi")   -tab을 해야 함
                 print("I love You")
                 # tab을 한 상태에서 값을 넣으면 love 함수의 내용으로 인식을 하게 된다.
             love() -> 함수 이름과 괄호를 사용함
             -> love 함수를 실행 한 결과가 나온다.
     * 객체(Object)

파라미터란?
기존에 구현한 함수를 변경해서 사용할 수 있다.
def love(name):   -> "name"이 파라미터  
    print("Hi")
    print(name)
    print("I love You")

love("Harin")  ->"Harin"이 파라미터 name에 적용이 된다.

-> <출력값>
    Hi
    Harin
    I love You
