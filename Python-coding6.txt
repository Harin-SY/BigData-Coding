변수 이해하기
 -python에서 x = x + 1에 사용되는 등호(=)는 x와 x + 1이 같다는 의미가 아니라 
 오른쪽에 있는 값을 왼쪽 변수에 대입하라는 "지정 연산자"를 말한다.
 
함수의 실행 순서
1)
def hi():                                   -> 함수를 정의하는 줄이라 어떠한 실행도 하지 않음
    print("Hi~")
    print("Welcom to Python!")     -> 여기까지 함수 정의

print("함수를 호출하기 전")          -> "함수를 호출하기 전" 출력
hi()                                         -> "hi"라는 함수를 실행시키는 명령어로 2개의 print() 값이 출력됨
print("함수를 호출한 뒤")             -> "함수를 호출한 뒤" 출력


2)
def sum(x):                           ->함수를 정의함
    return x + x                      ->함수 정의 종료

print("함수를 호출하기 전")      ->"함수를 호출하기 전" 출력
sum(sum(2) + sum(3))            -> sum(x)에 2가 대입되어 2 + 2를 통해 4, 3이 대입되어 3 + 3으로 6의 값이 되어
                                             4 + 6으로 10이 출력된다. 
print("함수를 호출한 뒤")         ->"함수를 호출한 뒤" 출력


return문 이해하기
return문은 함수가 "어떠한 값을 돌려주는 것"을 의미한다.
               -> 이외에도 함수를 즉시 종료시키기도 가능하다.
1)
def sum(x):                   ->함수 정의
    print("함수 시작")      
    return x + x
    print("함수 종료")        ->함수 정의 종료

print(sum(5))                    ->"함수 시작" 출력
                                         sum(x)에 5가 대입되어 5 +5 = 10이 되어 10을 출력함
                                         *return은 값을 출력도 하지만 종료시키는 기능이 있어
                                          print("함수 종료")는 출력되지 않음 - 의미 없는 코드("Dead Code")
print("Welcom to Python!")  -> "Welcom to Python!" 출력
